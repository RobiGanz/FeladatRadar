# FeladatRadar

A **FeladatRadar** egy feladatkezelő webalkalmazás, amely segít nyomon követni a feladatokat, órarendet és csoportokat.

## Funkciók

* **Feladatkezelés** – A diák felhasználó saját feladatokat adhatnak hozzá, teljesítettnek jelölhetik vagy törölhetik azokat
* **Csoportok** – Felhasználók csoportokat hozhatnak létre, tagokat hívhatnak meg e-mail alapján, és csoporton belül közösen kezelhetik a feladatokat és órarendet
* **Órarend** – Csoportonkénti órarend nyilvántartás
* **Tantárgyak** – Tantárgyak vagy Teendők kezelése csoportokon belül
* **Szerepkörök** – Kettő szerepkör: **Diák**, **Tanár** eltérő jogosultságokkal
* **Hitelesítés** – JWT alapú bejelentkezés és regisztráció

## Technológiák

|Réteg|Technológia|
|-|-|
|Backend|ASP.NET Core 8 (C#)|
|Adatbázis|Microsoft SQL Server|
|ORM|Dapper|
|Auth|JWT Bearer token|
|Frontend|Blazor (.NET)|
